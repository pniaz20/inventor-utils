SoXt 1.2.1 Released
===================

SoXt 1.2.1 is a micro release.  It was released to fix a documentation
generation problem with the prior release, and contains no code changes
or anything else.


NEWS File Excerpt:
------------------

New in SoXt 1.2.1 (2004-09-24):
* Did a doxygen razzia, fixed a doc build problem


Download Information:
---------------------

Source Code by FTP:
  ftp://ftp.coin3d.org/pub/coin/src/SoXt-1.2.1.tar.gz

We do not provide pre-compiled binary packages of the SoXt library
for any platform.
